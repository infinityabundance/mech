# mech

Rust workspace for reduced-order mechatronic architecture studies.

The primary deliverable in this repository is [`crates/mech-sim`](/home/one/mech/crates/mech-sim/README.md), a deterministic proof-of-life simulator for the paper:

`Gigawatt-Class Terrestrial Legged Vehicles: A Nuclear-Thermal, Pulse-Power, and Electrohydraulic Systems Architecture`
